# Bullish Momentum Burst Guide — Section 6

### Dollar Breakout: The Complement to 4% Breakout

**The Origin:** Dan Zenger's idea — scan for stocks above $40 that are up $1+.

> "4% breakout is finding lower-priced stocks. Dollar breakout is for higher-priced stocks that won't show up in a 4% scan."

---

### Why Dollar Breakout?

| Benefit | Detail |
|---|---|
| **Lower stops** | Stocks up 1–2% = stops at 2–2.5% with half-range stop |
| **Longer moves** | 8–12+ day moves (vs. 3–5 days for 4% breakout) |
| **Lower volatility** | Higher-priced stocks are inherently less volatile |
| **More capital efficient** | Tighter stops = larger position size per trade |
| **Slower, less crowded** | Next-day entries possible; no gap-up scramble |

> "Because most of these stocks breaking out with 1%, 1.5%, 2% on a breakout day, it allows you to get a very low risk entry."

---

### Who Benefits Most?

- **Larger accounts** — can deploy more capital per position
- **Working professionals** — next-day entry is viable
- **Lower-risk traders** — want 2–2.5% stops, not 5–8%

---

### TC2000 Scan Setup

**Bullish Dollar Breakout:**
```
C - O  >= 0.90       // Stock up $0.90+ from open
V >= 100,000         // Minimum liquidity
```

**Bearish Dollar Breakout:**
```
O - C >= 0.90        // Stock down $0.90+ from open
V >= 100,000
```

**Eliminate 4% overlap (to see only dollar breakouts):**
1. Run dollar breakout scan → copy to blank watchlist
2. Run 4% breakout scan → flag symbols
3. In dollar watchlist, remove flagged symbols
4. Remaining = pure dollar breakouts ($ up but < 4%)

> "Dollar breakout allows you to get into higher priced stocks with a lower risk compared to stocks which break out with 3%, 4% or more."

---

### Chart Template Setup

Use the same Two Lynch template as 4% breakout, but change the PCF conditions to:
- Dollar bullish: green spikes
- Dollar bearish: red spikes

**Differentiate 4% vs. dollar tabs with different background colors:**
- 4% breakout template → one color
- Dollar breakout template → different color

> "I color these things with a different color so that you know quickly this is a different color tab."

---

### Two Lynch Applies Equally to Dollar Breakout

| Check | Dollar Breakout |
|---|---|
| **Not up two days in a row** | Yes |
| **Linearity in first leg** | Yes |
| **Young setup** | First/second leg |
| **Narrow/negative day prior** | Ideally |
| **Consolidation quality** | Compact, orderly, no 4% breakdown |
| **Close near high** | Yes |

Same exact framework — only the breakout magnitude differs.

---

### Key Principle

> "Dollar breakout is something which is a very good method, and it is more suitable for people who have larger account and people who are working."
