**TC2000’s “Worden Common Stock” (or “US Common Stocks”) universe** is a pre-built, dynamically maintained watchlist of roughly 6,500+ stocks. It includes common stocks listed on the NYSE, NASDAQ, and NYSE American (AMEX), while explicitly excluding:

- ETFs
- Indices
- Preferred stocks (they have a separate watchlist)
- OTC/pink sheets/bulletin board stocks
- Canadian symbols
- Most non-equity instruments (warrants, rights, etc.)

It is the default broad “all investable common stocks” universe that TC2000 users (and many scanners/videos) rely on for EasyScans and PCF formulas. It is **not** a simple “all US stocks” list — Worden curates it for liquidity/listing status and actively maintains it.

### Why “Exact” Replication Is Challenging
TC2000’s list is proprietary/curated (they decide inclusion based on exchange listing status, data availability, and internal rules). There is no public downloadable CSV or official definition beyond the exclusions above. New listings, delistings, and edge cases (e.g., when a stock moves from OTC to major exchange) create minor timing differences.

**The closest high-quality public approximation** uses **Financial Modeling Prep (FMP)** data, which has excellent coverage of US equities with clean flags for ETF/fund/ADR/active status. FMP’s universe will be extremely close (often within a few hundred symbols) for practical scanning, backtesting, or building TC2000-style systems.

### Best Method: FMP Company Stock Screener
Use the **Company Stock Screener** endpoint:

**Endpoint**: `https://financialmodelingprep.com/stable/company-screener`

**Key parameters** (combine as needed):

| Parameter              | Recommended Value          | Purpose                                      | Notes |
|------------------------|----------------------------|----------------------------------------------|-------|
| `country`              | `"US"`                     | US-listed stocks                             | Core filter |
| `isEtf`                | `false`                    | Exclude ETFs                                 | Critical |
| `isFund`               | `false`                    | Exclude mutual funds, closed-end funds, etc. | Critical |
| `isActivelyTrading`    | `true`                     | Only actively traded                         | Critical for relevance |
| `exchange`             | `"NYSE"`, `"NASDAQ"`, or `"AMEX"` | Major exchanges only                  | Use multiple calls or omit for broader US |
| `isAdr`                | `false` (optional)         | Exclude ADRs if desired                      | See nuance below |
| `marketCapMoreThan`    | e.g. `1000000` or omit     | Basic liquidity filter                       | TC2000 includes micro-caps |
| `limit`                | `1000` or higher if supported | Pagination control                     | Test your tier |
| `includeAllShareClasses` | `true` (optional)        | Include multiple share classes               | Rarely needed |

**Response fields** you’ll get (very useful):
- `symbol`, `companyName`, `exchange`, `exchangeShortName`
- `country`, `isEtf`, `isFund`, `isActivelyTrading`, `isAdr`
- `sector`, `industry`, `marketCap`, `price`, `volume`, etc.

This combination closely mirrors TC2000’s criteria.

### Preferred Stocks Handling (Extra Step)
FMP does not have a direct `isPreferred` filter in the screener. Preferred stocks are usually excluded from the common stock universe in TC2000.

**Post-filter options**:
1. Filter `companyName` to exclude strings containing `"Preferred"`, `" Pref"`, `" PR "`, or similar.
2. For higher precision, fetch the company profile (`/stable/profile/{symbol}`) for additional classification or simply accept a very small number of edge cases (preferreds are a tiny fraction of the universe).

### ADR Nuance
- TC2000’s US Common Stocks **likely includes** sponsored ADRs listed on major US exchanges (they trade and chart like common stocks).
- FMP flags them with `isAdr: true`.
- **Recommendation for closest match**: Include them (omit or set `isAdr` filter to `null`/omit). If you want “domestic only,” add `isAdr: false`.

### Practical Python Implementation
```python
import requests
import pandas as pd
from typing import List, Dict

API_KEY = "YOUR_FMP_API_KEY"
BASE_URL = "https://financialmodelingprep.com/stable/company-screener"

def get_tc2000_like_universe(
    exchanges: List[str] = ["NYSE", "NASDAQ", "AMEX"],
    exclude_adrs: bool = False,
    min_market_cap: int = None
) -> pd.DataFrame:
    all_stocks = []
    
    for exch in exchanges:
        params = {
            "apikey": API_KEY,
            "country": "US",
            "isEtf": "false",
            "isFund": "false",
            "isActivelyTrading": "true",
            "exchange": exch,
            "limit": 1000
        }
        if exclude_adrs:
            params["isAdr"] = "false"
        if min_market_cap:
            params["marketCapMoreThan"] = min_market_cap
        
        response = requests.get(BASE_URL, params=params, timeout=30)
        if response.status_code == 200:
            data = response.json()
            if isinstance(data, list):
                all_stocks.extend(data)
        else:
            print(f"Error for {exch}: {response.status_code}")
    
    df = pd.DataFrame(all_stocks)
    
    # Optional: Clean preferred stocks by name
    if not df.empty:
        preferred_mask = df['companyName'].str.contains(
            r'Preferred| Pref\.| PR |Preferred Stock', 
            case=False, 
            na=False
        )
        df = df[~preferred_mask]
    
    # Deduplicate if any overlap
    df = df.drop_duplicates(subset=['symbol'])
    
    return df[['symbol', 'companyName', 'exchange', 'exchangeShortName', 
               'country', 'isAdr', 'sector', 'industry', 'marketCap']]

# Usage
universe = get_tc2000_like_universe(exclude_adrs=False)  # Closest to TC2000
print(f"Universe size: {len(universe)}")
print(universe.head())
```

You can save this to CSV/Parquet or load it into your backtester/database.

### Alternative / Complementary FMP Endpoints
- **Stock List** (`/stable/stock-list`): Broader unfiltered list. Good base if you want to apply your own logic.
- **Actively Trading List** (`/stable/actively-trading-list`): Another solid starting point with exchange/region filters.
- **Company Profile** (`/stable/profile/{symbol}`): Enrich with `isAdr`, description, IPO date, etc.
- **ETF List** (`/api/v3/etf-list`): Explicit ETF exclusion list if needed.

### Edge Cases & Considerations
- **Very small/micro caps & recent IPOs**: TC2000 includes many; FMP’s `isActivelyTrading` + major exchanges does too. Add a low `marketCapMoreThan` or none.
- **Share classes**: Rare duplicates — use `includeAllShareClasses` if relevant.
- **Data freshness**: FMP updates frequently (real-time quotes + fundamentals). TC2000 is also real-time for its users.
- **Coverage differences**: Expect 6,000–7,500+ symbols depending on exact filters and date. Minor variance from TC2000 is normal and acceptable for most use cases.
- **Rate limits & cost**: FMP has generous free tier and paid plans. Batch or cache results.
- **If you have TC2000 access**: Export the US Common Stocks watchlist (as CSV or via their tools) and cross-reference tickers with FMP for a near-perfect hybrid.

### Summary Recommendation
For **exact-as-possible** replication with FMP data:
1. Use the Stock Screener with `country=US`, `isEtf=false`, `isFund=false`, `isActivelyTrading=true` + major exchanges.
2. Post-filter preferred stocks by name (or profile).
3. Include ADRs for closest behavioral match to TC2000.
4. Add liquidity/volume filters if your strategy needs them (TC2000 users often do this downstream).

This gives you a clean, queryable, API-friendly universe that behaves like TC2000’s Worden Common Stock list for scanning, backtesting, or building custom tools — with the added benefit of FMP’s rich fundamentals, historical data, and profiles.
