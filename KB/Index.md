## How to Use This KB

1. **Have a specific question?** → Go directly to the relevant KB note (e.g., "Where do I put my stop?" → [[Stops]])
2. **Want to study a setup deeply?** → Read the curated guide (e.g., [[16. DEP Guide|DEP]])
3. **Want the raw source?** → Follow wikilinks from guides to [[../transcripts/|transcripts]]
4. **New to the system?** → Start with [[Entries]], [[Stops]], and [[Market Monitor]]
5. **Building a scanner?** → See [[Scans and Filters]] for FMP API mapping

## How to Navigate

- **KB notes** have comparison tables (all setups side-by-side)
- **Curated guides** have deep explanations of ONE setup
- **Transcripts** have the exact words Pradeep said (most comprehensive)
- Use **Obsidian graph view** to see how concepts connect
